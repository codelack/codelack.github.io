Fictional User Interface Desktop

2014-2015
Category: Websites & UI
Role: Designing, Animating
Used: Photoshop, After Effects

I am fascinated with the video, tech, and art field of FUI. This means many things: Fictional, Fake, Fantasy, or Futuristic User Interface. It is used in movies and television as fictional replacements of Operating Systems and the like. This is the start of my study in the subject.

I first designed three screens of what a Windows desktop would look like if I liked it in spring of 2014 for fun. I then turned it into a very simple animation in spring 2015. A cursor moves to show how it would be used on a desktop machine.

While I did design the screens, I used many icons found on deviantArt.com, as well as designed a few simple ones of my own (arrows, the Windows logo as I would envision it).
